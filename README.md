# 🪟 POS ร้านอลูมิเนียม-กระจก-สแตนเลส

ระบบ POS (Point of Sale) สำหรับร้านขายอุปกรณ์งาน:
- อลูมิเนียมโปรไฟล์ / ประตู-หน้าต่าง
- กระจก
- ท่อและอะไหล่งานรั้วสแตนเลส
- ฟิตติ้งงานเฟอร์นิเจอร์
- อุปกรณ์ผ้าม่าน

ใช้ **Supabase** เป็นฐานข้อมูล และ **GitHub Pages** เป็น Host

---

## 🚀 ขั้นตอนติดตั้ง

### ขั้นตอน 1 — สร้างโปรเจค Supabase

1. ไปที่ [supabase.com](https://supabase.com) → **New Project**
2. ตั้งชื่อโปรเจค เช่น `pos-alu-glass`
3. ตั้งรหัสผ่านฐานข้อมูล (จดไว้ด้วย)
4. เลือก Region ที่ใกล้ที่สุด (แนะนำ `Southeast Asia`)
5. รอสักครู่ให้โปรเจคสร้างเสร็จ

> ✅ Supabase Free Tier รองรับ **2 โปรเจคพร้อมกัน** — ถ้ามีโปรเจคอื่นอยู่แล้ว 1 โปรเจค ก็เพิ่มได้อีก 1 ฟรี โดยไม่กระทบกัน

---

### ขั้นตอน 2 — รัน SQL สร้างฐานข้อมูล

1. ในหน้า Supabase Dashboard → เลือก **SQL Editor**
2. คลิก **New Query**
3. เปิดไฟล์ `schema.sql` แล้วคัดลอกเนื้อหาทั้งหมด
4. วางใน SQL Editor แล้วคลิก **Run**
5. ✅ ระบบจะสร้างตารางและใส่ข้อมูลตัวอย่างให้อัตโนมัติ

---

### ขั้นตอน 3 — คัดลอก API Keys

1. ใน Supabase Dashboard → **Settings** → **API**
2. คัดลอก:
   - **Project URL** (เช่น `https://abcdefgh.supabase.co`)
   - **anon public** key (ยาวประมาณ 200 ตัวอักษร)
3. เปิดไฟล์ `config.js` แล้วแก้ไข:

```js
export const SUPABASE_URL  = 'https://YOUR-PROJECT-ID.supabase.co';
export const SUPABASE_ANON_KEY = 'eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...';
```

### ขั้นตอน 5 — เปิดใช้งาน GitHub Pages

1. ไปที่ GitHub repo ของคุณ → **Settings** → **Pages**
2. Source: **Deploy from a branch**
3. Branch: `main` / `/ (root)`
4. คลิก **Save**
5. รอ 1-2 นาที จะได้ URL เช่น `https://yourname.github.io/pos-alu/`

## 🔒 ความปลอดภัย

ระบบนี้ใช้ Supabase **anon key** พร้อม Row Level Security (RLS) ที่เปิดให้ทุกคนอ่าน-เขียนได้
เหมาะสำหรับการใช้งานภายในร้านที่ไม่ต้องการ login

หากต้องการความปลอดภัยเพิ่ม:
- ใช้ **Supabase Auth** (email/password) แล้วเปลี่ยน policy จาก `anon` เป็น `authenticated`
- หรือตั้ง IP Restriction ใน Supabase Dashboard → Settings → API → Restrictions

## 🆕 มีอะไรใหม่ในเวอร์ชันนี้

- **รายการสินค้าหน้าขายเป็นตาราง** — แสดง รหัสสินค้า / ชื่อ / หน่วยนับ / ราคา / สต๊อก แตะแถวเพื่อเพิ่มลงตะกร้า
- **ช่องรับเงิน** — พิมพ์ตัวเลขได้อิสระ (รองรับแป้นตัวเลขบนมือถือ)
- **ระบบลูกค้า + ยอดหนี้** — แท็บ "ลูกค้า/หนี้" ตั้งวงเงินเครดิตรายบุคคล ระบบบันทึกหนี้อัตโนมัติเมื่อขายแบบเครดิต และมีปุ่มชำระหนี้พร้อมประวัติ
- **หมวดหมู่ปรับเองได้** — ไปที่ "ตั้งค่า → หมวดหมู่สินค้า" เพิ่ม/แก้ไข/ลบ และเลือกสีป้ายได้

## ❓ คำถามที่พบบ่อย

**Q: ใช้ได้กี่เครื่องพร้อมกัน?**  
A: ได้ไม่จำกัด เพราะข้อมูลอยู่บน Supabase Cloud

**Q: ถ้าอินเทอร์เน็ตหลุดขณะขาย?**  
A: ขณะนี้ต้องใช้อินเทอร์เน็ต (รองรับ offline mode ในอนาคต)

**Q: Supabase Free Tier มีข้อจำกัดอะไร?**  
A: Storage 500MB, 50,000 rows, 2GB bandwidth/เดือน — เพียงพอสำหรับร้านค้าขนาดเล็ก-กลาง

**Q: ข้อมูลจะหายไหมถ้าไม่ได้ใช้นาน?**  
A: Supabase Free Tier จะ pause โปรเจคถ้าไม่มีกิจกรรมนาน 7 วัน แต่ข้อมูลยังอยู่ครบ เปิดใช้งานได้ปกติ
